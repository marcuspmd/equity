# Equity: Celebrating Invisible Heroes of Technology

> *A 2026 WeCoded Challenge Submission*
>
> Because true progress is built on diversity, and every individual has the power to change the world.

## About This Project

**Equity** is an interactive timeline experience dedicated to celebrating the overlooked contributions of women, LGBTQ+ individuals, people of color, and other marginalized groups in the evolution of technology.

This project is our submission to the [2026 WeCoded Challenge](https://dev.to/devteam/2026-wecoded-challenge-celebrating-gender-equity-in-dev-60j8) under the **"Frontend Art: Gender Equity"** prompt, showcasing how diversity has been fundamental to technological progress.

### The Problem We're Addressing

For too long, the narratives of tech have been narrow and exclusive. While countless individuals from underrepresented communities have shaped the digital world we know today, their stories remain largely unknown. Educational systems and media rarely highlight these contributions, creating a false narrative of who "belongs" in tech.

### Our Solution

We don't fight for equality anymore—we **showcase it**. We believe that:

- **Every person has inherent worth and individuality**
- **Technology was built by people of all races, colors, ethnicities, genders, and backgrounds**
- **Recognition of these contributions is not just right—it's essential**
- **Visibility inspires the next generation of changemakers**

## What You'll Find Here

This interactive experience highlights transformative figures across different domains:

- **🎯 Pioneers**: Early visionaries who laid the foundation for modern computing
- **💻 Early Coders**: Women and minorities who wrote the code that shaped our world
- **🚀 Space & Science**: Leaders who reached for the stars and beyond
- **⚖️ Civil Rights & Justice**: Advocates who fought for equality in tech spaces
- **🌍 Modern Era**: Contemporary leaders pushing boundaries today
- **👥 Communities**: Grassroots movements and collective efforts for change

## Getting Started

### Prerequisites
- Node.js (v18 or higher)

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd equity
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to the provided local URL

## Building for Production

### Local Build
```bash
npm run build
```

The built files will be in the `dist/` directory.

### Automatic Deployment

This project uses GitHub Actions to automatically build and deploy to GitHub Pages on every push to `main` or `master`.

**Setup Instructions:**

1. Enable GitHub Pages in your repository settings:
   - Go to Settings → Pages
   - Select "Deploy from a branch"
   - Choose `gh-pages` branch as source

2. The workflow will automatically:
   - Run on every push to `main` or `master`
   - Install dependencies
   - Build the project
   - Deploy to GitHub Pages

Your site will be available at: `https://your-username.github.io/equity/`
